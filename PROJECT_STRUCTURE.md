## üîß BACKEND FILES

### Main Backend Files
1. **Server (API)**: `/app/backend/server.py`
   - Main FastAPI application with all API endpoints
   - Handles authentication, products, cart, orders, payments
   - Over 500 lines of code

2. **Configuration**: `/app/backend/config.py`
   - Company details (UPI, Bank, GST, Phone, WhatsApp)
   - Pricing rules, GST rates, minimum order quantities
   - Easy to edit company information

3. **Environment Variables**: `/app/backend/.env`
   - Database connection (MONGO_URL, DB_NAME)
   - JWT secret key
   - CORS settings

4. **Dependencies**: `/app/backend/requirements.txt`
   - List of all Python packages required
   - Auto-generated by pip freeze

### Backend Utilities
5. **GST Validator**: `/app/backend/utils/gst_validator.py`
   - Validates GST number format
   - Validates order quantities (minimum 100 bags, multiples)

6. **Invoice Generator**: `/app/backend/utils/invoice_generator.py`
   - Generates PDF invoices with GST compliance
   - Includes company logo, customer details, pricing breakdown
   - Creates professional tax invoices

7. **Notifications**: `/app/backend/utils/notifications.py`
   - WhatsApp notification message templates
   - Email notification functions
   - Order status messages (placed, payment received, driver assigned, delivered)

## üìù HOW TO EDIT FILES


```bash

```

## üß™ HOW TO TEST

